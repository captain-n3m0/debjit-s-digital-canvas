# Debjit Naskar Portfolio

Personal cybersecurity portfolio for Debjit Naskar, built with Vite, React,
TypeScript, shadcn/ui, Tailwind CSS, GSAP, Lenis, and Three.js.

## Local Development

Install dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```

## Project Structure

- `src/pages/Index.tsx` assembles the portfolio landing page.
- `src/components/portfolio/` contains the custom animated portfolio sections.
- `src/components/ui/` contains reusable shadcn/ui primitives.
- `src/index.css` defines the global design tokens and utility classes.
